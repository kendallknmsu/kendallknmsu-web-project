# kendallknmsu-web-project
## Project 2 - HTML Foundation Page

**Theme:**
This website is based on the Salkantay Trek in Peru. It is designed as a blog-style page to document the hiking experience and provide information about the trek.

**What I Learned:**
I learned how to structure a basic HTML webpage using elements such as headings, paragraphs, lists, images, tables, and links. I also learned how to connect multiple pages using navigation links.

**Challenges:**
One challenge I faced was correctly linking pages between folders. I also had to experiment with HTML styling without using CSS. I also had trouble getting my image to load for some reason.
